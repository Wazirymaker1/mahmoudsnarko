## ✨ المميزات

| الميزة | الوصف |
|--------|-------|
| 🎯 | **Persistent Bombing** - لا يتم تخطي أي رقم، الهجوم مستمر حتى النجاح |
| 🔄 | **Smart Proxy Rotation** - تغيير تلقائي للبروكسي عند Rate Limit |
| 🎲 | **Full Randomization** - Device ID و User Agent مختلفين لكل طلب |
| 📁 | **Multi-Number Support** - دعم ملفات الأرقام (phones.txt) |
| 🌍 | **Proxy Management** - Auto/Manual Proxy مع اختبار تلقائي |
| 🤖 | **UA Control** - Random / Specific / Rotate modes |
| 💾 | **Results Saving** - حفظ النتائج في ملفات JSON |
| 📊 | **Wide Mode Display** - عرض النتائج في جدول منظم |
| 🔐 | **Subscription System** - نظام اشتراك لحماية الأداة |

⚙️ أوامر القائمة

الخيار الوصف
1 SPAM MODE - Single Number (هجوم لا يتوقف على رقم واحد)
2 SPAM MODE - File (هجوم على جميع الأرقام في الملف)
3 Proxy Settings (إعدادات البروكسي)
4 UA Settings (إعدادات User Agent)
5 Show Stats (عرض النتائج المحفوظة)
6 Exit

---

📁 تنسيق ملف الأرقام

قم بإنشاء ملف phones.txt في نفس المجلد:

```
+201234567890
+201098765432
+201112223334
```

---

🎯 طريقة العمل

وضع الرقم الواحد:

· الأداة تضرب نفس الرقم بشكل مستمر
· عند Rate Limit → تغيير بروكسي + استمرار
· لا يوجد تخطي للأرقام

وضع الملف:

· يقرأ الأرقام من الملف
· لكل رقم عدد محاولات محدد
· حفظ النتائج الكاملة في ملفات JSON
